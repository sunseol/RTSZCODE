# ⚔️ 중세 왕국 전쟁 (Medieval RTS)

Three.js로 만든 중세 배경 3D 실시간 전략 게임입니다. 절차적 생성 로우폴리 그래픽으로 외부 에셋 없이 실행됩니다.

## 🎮 플레이 방법

### 실행
```bash
npm install
npm start
```
브라우저가 자동으로 열리지 않으면 `http://localhost:8080` 로 접속하세요.

### 조작법
| 조작 | 기능 |
|------|------|
| **좌클릭** | 유닛/건물 단일 선택 |
| **좌클릭 드래그** | 박스 영역 다중 선택 (군사 유닛 우선) |
| **우클릭** | 이동 / 공격 / 자원 채집 (타겟에 따라 자동 판단) |
| **마우스 모서리** | 카메라 스크롤 |
| **WASD / 방향키** | 카메라 이동 |
| **Q / E** | 카메라 회전 |
| **마우스 휠** | 줌 인/아웃 |
| **ESC** | 선택 해제 |
| **S** | 선택 유닛 정지 |
| **1-4 숫자키** | 건물 생산 단축키 |

### 게임 목표
**적의 성을 파괴하면 승리**, 아군 성이 파괴되면 패배합니다. 35초마다 적의 공격 웨이브가 몰려옵니다.

## 🏰 게임 요소

### 자원
- 🪙 **Gold (금)**: 금광에서 채집. 유닛/건물 생산에 사용
- 🍖 **Food (식량)**: 유닛 생산에 사용

### 유닛
| 유닛 | 역할 | 비용 |
|------|------|------|
| 🧑‍🌾 **주민 (Villager)** | 금 채집, 약한 전투 | 30 금 / 10 식량 |
| 🗡️ **기사 (Knight)** | 강한 근접 전투, 높은 체력 | 60 금 / 40 식량 |
| 🏹 **궁수 (Archer)** | 원거리 공격, 낮은 체력 | 50 금 / 25 식량 |

### 건물
- 🏰 **성 (Castle)**: 기지. 주민 생산 + **식량 자동 생산(+1.5/초)**. 파괴 시 패배
- ⚒️ **병영 (Barracks)**: 기사/궁수 훈련 (🪙120 건설)
- 🌾 **농장 (Farm)**: **식량 대량 생산(+3/초)**. 유닛 생산 불가 (🪙80 건설)

### 자원
- 🪙 **Gold**: 금광 채집으로 획득. 건물 건설/유닛 생산에 사용
- 🍖 **Food**: **성과 농장에서 자동 생산**. 유닛 생산에 사용
  - 성만: +1.5/초 · 농장 추가 시: +3/초 · 농장을 여러 개 지을수록 식량 수입 증가

### 건설 시스템
1. 주민 선택 → 우측 하단 건설 버튼(또는 **B** = 병영, 향후 추가 단축키) 클릭
2. 반투명 ghost 건물이 마우스를 따라다님 (초록=건설 가능, 빨강=불가능)
3. 건설 위치 좌클릭 → 가까운 주민이 이동해서 건설
4. **건설 중 효과**: 건물이 투명한 상태에서 지면 아래에 있다가, 주민이 건설할수록 점점 나타나며 올라옴
5. 건설 완료 후 주민은 자동으로 금 채집으로 복귀

### 추천 전략
1. 시작 주민 3명이 자동으로 금을 캡니다 → 금이 모이면 성에서 주민 추가 생산
2. 금 120 모이면 **병영 건설** (B 키) → 기사/궁수 생산
3. 군사 유닛이 많아지면 **농장 건설**로 식량 수입 증가 (기사는 식량 40 소모)
4. 기사/궁수 부대 편성 후 적 성으로 진격
5. 웨이브 타이머를 주시하며 방어와 공격의 균형 유지

## 🛠️ 기술 스택
- **Three.js** (WebGL 3D 렌더링)
- **바닐라 JavaScript** (ES 모듈)
- 절차적 메시 생성 (나무, 바위, 건물, 유닛 모두 코드로 생성)
- importmap 방식 모듈 로딩

## 📁 프로젝트 구조
```
├── index.html              # 진입점 + UI 오버레이
├── style.css               # HUD 스타일
├── package.json
└── src/
    ├── main.js             # 부트스트랩
    ├── Game.js             # 메인 게임 클래스 + 루프
    ├── world/              # 지형, 조명
    ├── camera/             # RTS 카메라 컨트롤
    ├── entities/           # 유닛/건물 엔티티
    ├── systems/            # 선택/전투/자원/AI/입력 시스템
    ├── ui/                 # HUD, 생산 메뉴
    └── utils/              # 헬퍼 함수
```
